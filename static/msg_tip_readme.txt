# 消息提示音说明

为了使uni-im消息提醒功能正常工作，请将消息提示音文件上传到云存储，而不是放在static目录下。系统会自动将音频文件缓存到本地，提高后续播放的响应速度。

## 配置步骤
1. 您可以从免费音效网站下载合适的消息提示音
2. 确保文件格式为MP3
3. 将文件命名为'msg_tip.mp3'并上传到您的云存储服务
4. 复制文件的完整URL地址
5. 打开 `uni_modules/uni-im/sdk/init/msgEvent.js` 文件
6. 将 `https://your-cloud-storage-domain.com/msg_tip.mp3` 替换为您的实际云存储文件URL

## 功能说明
- 系统会检查本地是否已缓存音频文件，如果存在则直接播放本地版本
- 如果本地没有缓存，则从云端下载并保存到设备的临时目录中
- 首次使用时需要从云端下载，后续使用将使用本地缓存版本，提高响应速度
- 即使下载失败，系统也会尝试直接播放云端文件作为降级方案

## 注意事项
- 文件大小建议控制在几百KB以内，避免影响加载速度
- 提示音长度建议在1-3秒之间，不宜过长
- 确保音效文件有合法的使用权限
- 确保云存储文件的访问权限设置为公开可读
- 如果使用uniCloud，可以将文件上传到云存储并使用云函数获取访问链接
